# JAVA_SPring-CheatSheet 
# Create a JavaSpring Project

#1 Install Apache Maven
-To install Maven, first needs to install HomeBrew. Download HomeBrew at https://brew.sh/. 
-In Terminal 
- $ brew install maven
-Check maven version
- $ mvn version
